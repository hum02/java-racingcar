# 프로그램 규칙
- [ ] 모든 로직의 단위 테스트를 구현한다.
- [ ] 자바 코드 컨벤션을 지키면서 프로그래밍한다.
- [ ] 한 메서드에 오직 한 단계의 들여쓰기를 한다.
- [ ] else 예약어를 쓰지 않는다.
- [ ] 함수의 길이가 15라인을 넘어가지 않도록 구현한다.
- [ ] 기능 단위로 커밋한다.

# 기능목록
## 주요기능
- [ ] 입력받은 이름으로 자동차 생성하기
  - [ ] 생성 시, 이름이 5자 초과인 경우 다시 이름을 입력받기
- [ ] 랜덤한 숫자를 생성하기
- [ ] 차 전진/멈춤
- [ ] 우승자 계산

## 입력
- [X] 자동차 이름 입력받기
  - [x] 공백 입력 시 재입력받기
- [ ] 시도횟수 입력받기
  - [ ] 문자 or 공백 입력 시 재입력받기 

## 출력
- [ ] 차의 이동현황을 출력하기
